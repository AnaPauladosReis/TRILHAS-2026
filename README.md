# Estrutura de Dados - Exercícios Práticos

## 📚 Sobre o Repositório

Este repositório contém a resolução completa dos exercícios práticos da trilha de **Estrutura de Dados**. Cada módulo aborda conceitos fundamentais e avançados, com implementações em JavaScript focadas em clareza, eficiência e boas práticas.

**Autora:** Ana Paula dos Reis  
**Ano:** 2026  
**Linguagem:** JavaScript 

---

## 📂 Estrutura do Repositório

O repositório está organizado por módulos, cada um contendo um arquivo JavaScript com a resolução completa dos exercícios:

| Módulo | Conteúdo | 
|--------|----------|
| 02 | Vetores, Listas Simples e Duplamente Encadeadas, Pilhas, Filas
| 03 | Recursividade - Fatorial, Fibonacci, Torres de Hanói, Percursos em Árvores 
| 04 | Árvores Binárias, AVL, Heaps, Grafos (DFS, BFS, Dijkstra, Floyd-Warshall)
| 05 | Tabelas Hash - Encadeamento, Endereçamento Aberto, Dicionário, Análise de Desempenho 
| 06 | Union-Find, Algoritmo de Kruskal, Árvores B e B+
